# FastAPI-Financial-Document-Management-with-Semantic-Analysis-
This project is a FastAPI-based backend system designed to manage and analyze financial documents using AI-powered semantic search.  It allows users to upload documents, store metadata, and retrieve meaningful insights using a Retrieval Augmented Generation (RAG) pipeline powered by vector embeddings and LLMs.

## Features

- User Authentication using JWT Tokens  
- Role-Based Access Control (RBAC)  
- Upload and manage financial documents  
- Metadata based document search  
- Semantic search using Qdrant Vector Database  
- RAG pipeline using LangChain + Groq LLM  
- Reranking system for improved retrieval accuracy

## Tech Stack

- FastAPI  
- MySQL (SQLAlchemy)  
- LangChain  
- Qdrant (Vector Database)  
- HuggingFace Embeddings  
- Groq (LLM - LLaMA 3.1)  
- Python
